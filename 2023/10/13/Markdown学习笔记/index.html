Markdown学习笔记

2023/10/13 编辑 Markdown
1、标题的语法

“# title” 即为标题的语法。需要注意的是，”#” 与 title 之间存在一个空格。# 的数量表征标题的层级，数量越多，标题的级别越低，字体也越小。

2、正文部分

  1. 段落：使用空白行进行段落之间的分隔，不要使用 Tab 和 Space 控制缩进。

  2. 换行：一行末尾添加两个及以上空格，然后按回车键。

  3. 强调：

    1. 粗体——两个“星号”：My name is NFF.
    2. 斜体——一个“星号”：My name is NFF.
    3. 同时——三个“星号”：My name is NFF.

  4. 块引用：

    在段落前添加 > 可以创建块引用

    块引用可以嵌套使用

  5. 链接：[超链接显示名] (超链接地址 “超链接标题”)
    本文参考自Markdown官方教程
    使用<>可以使URL或email变成可点击的链接，如 https://baidu.com

  6. 添加图片：![图片alt] (图片链接 “图片标题”)

  7. 转义字符：\
