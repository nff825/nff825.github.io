1 模态分析的作用

1.1 确定自然频率

1.2 确定模态形状 （特征向量）

1.3 FE model integrity

2 模态分析的分类

根据边界条件分类：

2.1 自由（刚体运动）

2.2 固定

3 仿真流程

3.1 打开 HyperMesh，选择 Nastran

3.2 创建几何

创建两个点：Geom –> nodes，坐标分别为(0, 0,0) 与 (0,100,0)

3.3 划分网格

2D –> drag，选中上述两点，沿 x 轴，距离为 1000
2D –> autoMesh，选中网格，自动划分

3.4 设置材料

点击 Materials，创建材料 steel：isotropic、MAT1，点击 create/edit
设置材料三属性,，点击 return
E = 2.1e5
NU = 0.3
RHO = 7.89e-9

3.5 设置属性

点击 Properties，创建属性 sectional：2D、PSHELL、steel，点击create/edit
设置厚度 T = 5，点击 return

3.6 更新组件

点击 Components，点击 update
选中组件，点击 property，选择 sectional，点击 update，点击 return

3.7 清除临时节点

Geom –> temp nodes – > clear all –> return

3.8 设置载荷

点击 Load Collectors，创建载荷 eigrl：EIGRL，点击 create/edit
设置 ND = 10，点击return

3.9 设置载荷步

Analysis –> loadsteps
创建载荷步 modal：type 选择 normal modes、METHOD(STRUCT) 选择 eigrl，点击 create
点击 edit，勾选 OUTPUT，勾选 DISPLACEMENT，勾选 SORTING 和 FORMAT，点击 return

3.10 设置

Analysis –> control cards
点击 SOL：Analysis 选择 Normal Modes，点击 return 后
点击 PARAM：勾选 POST，设置 POST_V1 = -1，点击 return

3.11 导出求解文件

File –> export –> Solver Deck
选择 Nastran，导出 bdf 文件

3.12 固定情形

3.12.1 设置边界条件

点击 Load Collectors，创建边界条件 spc：no card image，点击 create，点击 return
Analysis –> constrains
选中最左侧的节点，勾选 6 个自由度，load type 选择 SPC，点击 create，点击 return

3.12.2 设置载荷步

Analysis –> loadsteps
勾选 SPC，选择 spc，点击 update，点击 return

3.12.3 导出求解文件

File –> export –> Solver Deck
选择 Nastran，导出 bdf 文件

3.13 求解

打开 MSC Nastran
选择对应的 bdf 文件
点击 Run

3.14 后处理（可视化）

打开 HyperView
Load model
Load results
点击 Apply
