这种网络结构，如图1 ( b )所示，通过强制预测的各向异性张量位于各向同性张量的基础上，嵌入了旋转不变性。旋转不变性意味着流体流动的物理不依赖于观察者坐标系的方向。这是一个基本的物理原理，重要的是任何湍流闭合都要服从它。否则，机器学习模型对定义在不同方向的轴上的相同流进行评估，可能会产生不同的预测结果。
