深度学习 2023/10/19

神经网络的“引擎”：基于梯度的优化

对于第一个模型示例，每个神经层都对输入数据进行如下变换：

output = relu(dot(input, W) + b)

在这个表达式中，W 和 b 均为该层的属性，被称为该层的**权重 (weight) **或可训练参数 (trainable parameter)，分别对应属性 kernel 和 bias。这些权重包含模型从训练数据中学到的信息。

一开始，这些权重矩阵取较小的随机值，这一步叫作**随机初始化 (random initialization)。下一步则是根据反馈信号逐步调节这些权重。这个逐步调节的过程叫作训练 (training)**，也就是机器学习中的“学习”过程。

上述过程发生在一个训练循环 (training loop) 内，其具体流程如下。在一个循环中重复下列步骤，直到损失值变得足够小。

​ (1) 抽取训练样本 x 和对应目标 y_true 组成的一个数据批量。

​ (2) 在 x 上运行模型 (这一步叫作前向传播，forward pass)，得到预测值 y_pred。

​ (3) 计算模型在这批数据上的损失值，用于衡量 y_pred 和 y_true 之间的差距。

​ (4) 更新模型的所有权重，以略微减小模型在这批数据上的损失值。

loss_value = f(W)

张量 grad(loss_value, W0) —— f 在 W0 点的导数，其形状与W相同

每个元素 grad(loss_value, W0)[i, j] ——当 W0[i, j] 发生变化时 loss_value 变化的方向和大小

张量 grad(loss_value, W0) ——函数 f(W) = loss_value 在 W0 处的梯度，也叫作 “loss_value 相对于 W 在 W0 附近的梯度”

小批量随机梯度下降 (mini-batch stochastic gradient descent，小批量 SGD)

​ (1) 抽取训练样本 x 和对应目标 y_true 组成的一个数据批量

​ (2) 在 x 上运行模型，得到预测值 y_pred，这一步叫作前向传播

​ (3) 计算模型在这批数据上的损失值，用于衡量 y_pred 和 y_true 之间的差距

​ (4) 计算损失相对于模型参数的梯度，这一步叫作反向传播 (backward pass)

​ (5) 将参数沿着梯度的反方向移动一小步，比如 W -= learning_rate * gradient，从而使这批数据上的损失值减小一些，学习率 (learning_rate) 是一个调节梯度下降“速度”的标量因子

小批量 SGD 的变体

  1. 真 SGD (true SGD)：每次迭代只抽取一个样本和目标
  2. 批量梯度下降 (batch gradient descent)：每次迭代都在所有数据上运行
  3. 优化方法 (optimization method) 或优化器 (optimizer)：
    1. 带动量的 SGD、Adagrad、RMSprop 等
    2. 它们的不同之处在于，计算下一次权重更新时还要考虑上一次权重更新，而不是仅考虑当前的梯度值
    3. 动量解决了 SGD 的两个问题：收敛速度和局部极小值

反向传播算法 (backpropagation algorithm)

链式法则 (chain rule)

grad(y, x) == grad(y, x1) * grad(x1, x)

张量 (tensor)

0阶张量 (标量, scalar)

x = np.array(12)

1阶张量 (向量, vector)

x = np.array([1, 2, 3, 4, 5])

该张量有一个轴，该向量为5维向量

2阶张量 (矩阵, matrix)

x = np.array([[1, 2, 3, 4, 50],

​ [6, 7, 8, 9, 10],

​ [1, 2, 3, 4, 20])

该张量有两个轴 (行和列)

3阶张量与更高阶的张量

x = np.array([[[1, 2, 3, 10],

​ [2, 4, 6, 20]],

​ [[1, 3, 5, 10],

​ [2, 5, 8, 20]]])

该3阶张量由两个矩阵组成，可以理解为数字组成的长方体

关键属性

  1. 阶数 (轴的个数, ndim)

  2. 形状 (张量沿每个轴的维度大小或元素个数, shape)

    矩阵的形状为 (3, 5)

    向量的形状为 (5,)

    标量的形状为 ()

  3. 数据类型 (dtype)

  4. 以MNIST数据集为例：

    train_images.ndim = 3

    train_images.shape = (60000, 28, 28)

    train_images.dtype = unit8 8位整数, 取值为0~255

操作张量

张量切片 (tensor slicing)

沿着第一个轴操作张量

train_images[i] 第 i 张数字图像

my_slice = train_images[10:100] 选择第10~100 (不含100) 张数字图像，等同于

my_slice = train_images[10:100, : , : ]

my_slice = train_images[10:100, 0:28, 0:28]

沿着第二个和第三个轴操作张量

my_slice = train_images[ : , 14:, 14:] 所有图像的右下角14像素×14像素的区域

my_slice = train_images[ : , 7:-7, 7:-7] 所有图像中心的14像素×14像素的区域

数据批量

张量的第一个轴 (轴0) 是样本轴 (sample axis)，深度学习模型将数据拆分为小批量后进行处理

batch = train_images[:128] 第0个批量

batch = train_images[128 * n:128 * (n + 1)] 第 n 个批量

第一个轴 (轴0) 叫做批量轴 (batch axis)

现实世界中的数据张量实例

  1. 向量数据：2阶张量，形状为 (samples, features)
    • 每个样本是一个特征向量
  2. 时间序列数据：3阶张量，形状为 (samples, timesteps, features)
    • 每个样本是特征向量组成的序列，序列长度为 timesteps
  3. 图像数据：4阶张量，形状为 (samples, height, width, channels)
    • 每个样本是一个二维像素网格，每个像素由一个”通道” (channel) 向量表示
    • 一张灰度图像可以保存在一个2阶张量中
  4. 视频数据：5阶张量，形状为 (samples, frames, height, width, channels)
    • 每个样本是都是由图像 (一帧) 组成的序列，序列长度 (每个视频帧的数量) 为 frames

张量计算

逐元素运算 (加法和 relu 运算)

import numpy as np

z = x + y 逐元素相加

z = np.maximum(z, 0.) 逐元素 relu

z = x * y 逐元素乘积

广播 (broadcast)

在没有歧义且可行的情况下，阶数较低的张量会被广播，以匹配阶数较高的*张量的形状。步骤如下：

  1. 向小张量添加轴 (广播轴，broadcast axis)，使其 ndim 与大张量相同
  2. 将小张量沿着广播轴重复，使其形状与大张量相同

张量积 (tensoor product) 或点积 (dot product)

  1. 向量-向量点积

x = np.random.random((32,))

y = np.random.random((32,))

z = np.dot(x, y)

两个向量的点积是一个标量，只有元素个数相同的向量才能进行点积运算

  1. 矩阵-向量点积

    矩阵的第1维与向量的第0维必须相同，运算结果是一个向量，该向量的维数与矩阵第0维相同

    注意，只要两个张量中有一个的 ndim > 1，则dot运算不对称，即 dot(x, y) != dot(y, x)

  2. 矩阵-矩阵点积

    对于矩阵x和y，当且仅当 x.shape[1] == y.shape[0] 时，方可进行点积计算，点积结果为一形状为 (x.shape[0], y.shape[1])的矩阵，其元素是x的行与y的列之间的向量点积。

  3. 更高阶的张量点积运算

    (a, b, c, d) · (d,) —-> (a, b, c)

    (a, b, c, d) · (d, e) —-> (a, b, c, e)

张量变形 (tensor reshaping)

  • 转置 (transpose)

张量运算的几何解释

  • 加法：将物体沿着某个方向**平移 (translation) **一段距离
  • 与矩阵 R = [[cos (theta), -sin (theta)], [sin (theta), cos (theta)]] 点积：逆时针旋转 (rotation) θ 角
  • 与矩阵 R = [[horizontal_factor, 0], [0, vertical_factor]] 点积：在水平和垂直方向上进行缩放 (scaling)
  • 与任意矩阵点积：线性变换 (linear transform)
  • 一次点积与一次加法 (线性变换+平移)：仿射变换 (affine transform)

Markdown学习笔记
编辑 2023/10/13
1、标题的语法

“# title” 即为标题的语法。需要注意的是，”#” 与 title 之间存在一个空格。# 的数量表征标题的层级，数量越多，标题的级别越低，字体也越小。

2、正文部分

  1. 段落：使用空白行进行段落之间的分隔，不要使用 Tab 和 Space 控制缩进。

  2. 换行：一行末尾添加两个及以上空格，然后按回车键。

  3. 强调：

    1. 粗体——两个“星号”：My name is NFF.
    2. 斜体——一个“星号”：My name is NFF.
    3. 同时——三个“星号”：My name is NFF.

  4. 块引用：

    在段落前添加 > 可以创建块引用

    块引用可以嵌套使用

  5. 链接：[超链接显示名] (超链接地址 “超链接标题”)
    本文参考自Markdown官方教程
    使用<>可以使URL或email变成可点击的链接，如 https://baidu.com

  6. 添加图片：![图片alt] (图片链接 “图片标题”)

  7. 转义字符：\
