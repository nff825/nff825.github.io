1 静力分析

1.1 含义

计算在固定不变的载荷作用下结构的响应，不需要考虑惯性 (inertia) 和阻尼 (damping) 的影响

1.2 公式

$$F=[K][u]$$

  • K 是结构刚度 (stiffness)，$K=AE/L$
  • A 是横截面面积，E 是杨氏模量，L 是长度

1.3 理论计算结果

应力

$$\sigma=F/A$$

位移
$$d=PL/AE$$

2 流程

2.1 打开 HyperMesh，选择 Nastran

2.2 创建几何

Geom–>nodes–>create：(0, 0, 0)
2D–>planes–>square：选中该点，z-axis，mesh, dele surf，size=100

2.3 创建材料

Materials–>create–>steel：ISOTROPIC，MAT1
编辑，设置三属性

2.4 创建属性

Properties–>create–>pros：2D，PSHELL，steel–>create–>return
编辑，设置厚度为 5

2.5 更新组件

Components–>update–>选中该组件，pros–>update–>return

2.6 检查

Card Edit–>检查材料和属性
f10–>查看网格厚度

2.7 设置边界条件

Load Collectors–>create–>constrain–>create
Analysis–>constraints–>create–>选中最左侧的节点：勾选所有自由度，SOC–>create–>return

2.8 设置载荷

Load Collectors–>create–>load–>create
Analysis–>forces–>create–>选中最右侧的节点：18.18，x-axis–>create–>return

2.9 设置载荷步

2.9.1 选择载荷容器

Analysis–>loadsteps–>static：linear static，SPC=constrain，LOAD=load–>create

2.9.2 选择输出

Edit–>OUTPU–>DISPLACEMENT–>SORTING，FORMAT–>SPEFORCES–>SORTING，FORMAT–>STRESS–>SORTING，FORMAT–>return
SORT1：结果按“时间步”或“频率”排序

Time = 1.0
  Node 1: Displacement
  Node 2: Displacement
  ...
Time = 2.0
  Node 1: Displacement
  Node 2: Displacement
  ...

SORT2：结果按“节点”或“单元”排序

Node 1:
  Time = 1.0: Displacement
  Time = 2.0: Displacement
  ...
Node 2:
  Time = 1.0: Displacement
  Time = 2.0: Displacement
  ...

2.10 设置控制卡片

Analysis–>control cards

2.10.1 设置分析类型

SOL：Static

2.10.2 设置文件格式

PARAM：POST，-1
POST 控制后处理文件的格式（-1 表示求解后将结果输出到 op2 文件，0 表示输出到 xbd 文件）

2.11 导出求解文件

Export Solver Deck–>static_analysis.bdf–>Export
使用编辑工具，查看 bdf 文件

2.12 求解 bdf 文件

在 f06 文件中：搜索 fatal 与 epsilon

  • 存在 fatal 表示求解错误，可根据报错信息进行解决
  • 残差 epsilon 大于允许值 (如1e-9) ，表明模型刚度有奇异
  • 参考资料：https://zhuanlan.zhihu.com/p/710725697

2.13 后处理

使用 HypweView读取 op2 文件

3.1.1 位移

Contour–>Displacement
修改图例：Legend–>edit legend
放缩变形：Deformed–>Value

3.1.2 应力

Contour–>Stress
