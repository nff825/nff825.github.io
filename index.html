1 静力分析

1.1 含义

计算在固定不变的载荷作用下结构的响应，不需要考虑惯性 (inertia) 和阻尼 (damping) 的影响

1.2 公式

$$F=[K][u]$$

  • K 是结构刚度 (stiffness)，$K=AE/L$
  • A 是横截面面积，E 是杨氏模量，L 是长度

1.3 理论计算结果

应力

$$\sigma=F/A$$

位移
$$d=PL/AE$$

2 流程

2.1 打开 HyperMesh，选择 Nastran

2.2 创建几何

Geom–>nodes–>create：(0, 0, 0)
2D–>planes–>square：选中该点，z-axis，mesh, dele surf，size=100

Load Collectors–>create–>constrain–>create
Analysis–>constraints–>create–>选中最左侧的节点：勾选所有自由度，SOC–>create–>return

2.8 设置载荷

Load Collectors–>create–>load–>create
Analysis–>forces–>create–>选中最右侧的节点：18.18，x-axis–>create–>return

2.9 设置载荷步

2.9.1 选择载荷容器

Analysis–>loadsteps–>static：linear static，SPC=constrain，LOAD=load–>create

  • 残差 epsilon 大于允许值 (如1e-9) ，表明模型刚度有奇异

能量有限元(MATLAB)
2025/4/30

主程序

功能

指定参数空间（激励的位置、幅值、频率），进行能量有限元分析

代码

%% 获取数据 不同激励设置
clear; close all; clc
%% ==========创建数据存储文件夹 ==========
output_folder = 'EFEA_data_20250429_excitation_1800';
if ~exist(output_folder, 'dir')
    mkdir(output_folder);
end
%% ========== 设置固定参数 ==========
geometry.length_x = 0.5075;    % 薄板在 x 方向的长度
geometry.length_y = 0.4375;    % 薄板在 y 方向的长度
geometry.h = 42.4382e-3;       % 薄板的厚度

num_elements_x = 30;            % x 方向的单元数量
num_elements_y = 30;            % y 方向的单元数量
num_nodes = (num_elements_x+1)^2; % 节点数量

material.E = 1.6966e9;          % 弹性模量
material.rho = 94.2547;         % 密度
material.mu = 0.3;              % 泊松比
material.damping_ratio = 0.05; % 阻尼比
%% ========== 设置可变参数范围 ==========
frequencies = [500:10:550];
amplitudes = [1:0.1:1.5];
excitation_shifts = generate_excitation_shifts(num_nodes, 50);
COUNT = numel(frequencies) * numel(amplitudes) * numel(excitation_shifts);
%% ========== 生成数据 (并行计算) ==========
delete(gcp); % 关闭并行池
CoreNum = 4; % 设定机器CPU核心数量
parpool(CoreNum); % 开启并行池
parfor counter = 1:COUNT
    [frequency, F, offset] = get_parameters(counter, frequencies, amplitudes, excitation_shifts);

    %% ========== 激励设置 ==========
    excitation_freq = frequency;                    % 激励频率
    excitation_amp = F;                              % 激励幅值
    excitation_pos = round(num_nodes/2) + offset;    % 激励位置
    %% ========== 能量有限元分析 ==========
    output_filename = sprintf('%s/data_%d.txt', output_folder, counter);
    energy_density = EFEM_2D_plate(geometry, material, struct('freq', excitation_freq, 'amp', excitation_amp, 'pos', excitation_pos), num_elements_x, num_elements_y);
    %% ========== 写入参数数据 ==========
    fileID = fopen(output_filename, 'w');
    fprintf(fileID, '%.6f %.6f %.6f %.6f %.6f %.6f %.6f %.6f %.6f %.6f\n', ...
        excitation_freq, excitation_amp, excitation_pos, ...
        material.E, material.rho, material.mu, material.damping_ratio, ...
        geometry.length_x, geometry.length_y, geometry.h);
    %% ========== 换算能量密度（dB） ==========
    E = reshape(energy_density, num_elements_x+1, num_elements_y+1);
    E_ref = 1e-12;
    E(E < E_ref) = E_ref;
    E_dB = 20 * log10(E / E_ref);
    %% ========== 写入能量密度数据 ==========
    [x_coords, y_coords] = meshgrid(linspace(0, geometry.length_x, num_elements_x+1), linspace(0, geometry.length_y, num_elements_y+1));
    for i = 1:numel(x_coords)
        fprintf(fileID, '%.6f %.6f %.6f\n', x_coords(i), y_coords(i), E_dB(i));
    end
    fclose(fileID);
    fprintf('%d/%d | freq = %d amp = %d offset = %d | %s\n', counter, COUNT, frequency, F, offset, output_filename);
end

disp('done');

1 模态分析的作用

1.1 确定自然频率

1.2 确定模态形状 （特征向量）

1.3 FE model integrity

2 模态分析的分类

根据边界条件分类：

2.1 自由（刚体运动）

2.2 固定

3 仿真流程

3.1 打开 HyperMesh，选择 Nastran

3.2 创建几何

创建两个点：Geom –> nodes，坐标分别为(0, 0,0) 与 (0,100,0)

3.3 划分网格

2D –> drag，选中上述两点，沿 x 轴，距离为 1000
2D –> autoMesh，选中网格，自动划分

3.4 设置材料

点击 Materials，创建材料 steel：isotropic、MAT1，点击 create/edit
设置材料三属性,，点击 return
E = 2.1e5
NU = 0.3
RHO = 7.89e-9

3.5 设置属性

点击 Properties，创建属性 sectional：2D、PSHELL、steel，点击create/edit
设置厚度 T = 5，点击 return

3.6 更新组件

点击 Components，点击 update
选中组件，点击 property，选择 sectional，点击 update，点击 return

3.7 清除临时节点

Geom –> temp nodes – > clear all –> return

3.8 设置载荷

点击 Load Collectors，创建载荷 eigrl：EIGRL，点击 create/edit
设置 ND = 10，点击return

3.9 设置载荷步

Analysis –> loadsteps
创建载荷步 modal：type 选择 normal modes、METHOD(STRUCT) 选择 eigrl，点击 create
点击 edit，勾选 OUTPUT，勾选 DISPLACEMENT，勾选 SORTING 和 FORMAT，点击 return

3.10 设置

Analysis –> control cards
点击 SOL：Analysis 选择 Normal Modes，点击 return 后
点击 PARAM：勾选 POST，设置 POST_V1 = -1，点击 return

3.11 导出求解文件

File –> export –> Solver Deck
选择 Nastran，导出 bdf 文件

3.12 固定情形

3.12.1 设置边界条件

点击 Load Collectors，创建边界条件 spc：no card image，点击 create，点击 return
Analysis –> constrains
选中最左侧的节点，勾选 6 个自由度，load type 选择 SPC，点击 create，点击 return

3.12.2 设置载荷步

Analysis –> loadsteps
勾选 SPC，选择 spc，点击 update，点击 return

3.12.3 导出求解文件

File –> export –> Solver Deck
选择 Nastran，导出 bdf 文件

3.13 求解

打开 MSC Nastran
选择对应的 bdf 文件
点击 Run

3.14 后处理（可视化）

打开 HyperView
Load model
Load results
点击 Apply

  1. 与导师的沟通技巧
